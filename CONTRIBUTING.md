# Contributing

This repository is the **architecture narrative hub** for the governed AI reference stack.

- **Code changes** belong in the respective [vpeetla-ai](https://github.com/vpeetla-ai) org repos.
- **Architecture decisions** → add ADRs under `adr/` using the `ADR-0NN-slug.md` numbered format.
- **Case studies** → add under `case-studies/` with problem, architecture, trade-offs, impact, and live demo links.
- **Portfolio site** → [venkat-ai-portfolio](https://github.com/vpeetla-ai/venkat-ai-portfolio) consumes this content (wired in phase 2).

Keep employer-specific details generalized where required.
